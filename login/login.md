# TeamPortfolio Login Page

## Overview
This is the login page for TeamPortfolio, a professional team collaboration platform. The page features a modern, responsive design with a split-panel layout.

## Features

### Visual Design
- Split-panel layout with gradient backgrounds
- Modern, clean interface with smooth animations
- Responsive design for all screen sizes
- Professional typography using Inter and Poppins fonts

### User Interface Elements
1. **Left Panel**
   - Brand logo with "PRO" badge
   - Welcome message and subtitle
   - Feature highlights with icons
   - Back to home link (mobile only)

2. **Right Panel**
   - Login form with username/email field
   - Password field with visibility toggle
   - "Remember me" checkbox
   - Forgot password link
   - Demo credentials section
   - Social login buttons (Google, GitHub)
   - Sign up link
   - Back to home link

### Interactive Features
- Form validation with real-time error messages
- Password visibility toggle
- Demo credentials auto-fill
- Loading states and animations
- Notification system for user feedback
- Ripple effect on button clicks
- Keyboard shortcuts
- Responsive password reset modal

## Technical Details

### Form Validation
- Client-side validation for required fields
- Email format validation for password reset
- Real-time validation feedback
- Error state animations

### Authentication Simulation
- Demo credentials: `demo@teamportfolio.com` / `demo123`
- Session storage for login state
- Local storage for "remember me" functionality
- Simulated API delays for realistic UX

### Security Features
- Secure password input with visibility toggle
- Session-based authentication simulation
- No real credentials stored
- Frontend-only authentication for demo purposes

## Browser Support
- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## Dependencies
- Font Awesome 6.4.0 (icons)
- Google Fonts (Inter, Poppins)
- Vanilla JavaScript (no frameworks)
